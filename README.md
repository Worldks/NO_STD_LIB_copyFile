# NO_STD_LIB_copyFile
Копирует содержимое файла в другой файл. Программа не использует стандартную библиотеку языка Си.

#  Сборка и использование
В директории build 
  make prog
  Заполнить файл file_from некоторым содержимым
  ./prog file_from /*Имя файла в который будет происходить копирование*/
